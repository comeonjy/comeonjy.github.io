Docker 命令详解

发表于 2020-08-06 分类于 Tool 阅读次数： Valine：
本文字数： 1.3k 阅读时长 ≈ 1 分钟

容器生命周期管理

  • -d 后台运行容器并返回容器ID
  • -i 以交互模式运行容器
  • -t 为容器重新分配一个伪终端
  • -P 端口随机映射
  • -p 指定端口映射 宿主机端口:容器端口
  • –name | -n 为容器指定一个名称
  • -e server="dev" 设置环境变量
  • –env-file=[] 指定文件读入环境变量
  • -dns 8.8.8.8 指定容器使用的DNS，默认和宿主一致
  • -h 指定容器的hostname
  • -m 设置容器使用容器最大值
  • –net=“bridge” 指定容器的网络连接类型
  • –link=[] 添加链接到另一个容器
  • –volume，-v 绑定一个卷
  • -s 向容器发送一个信号
  • -f 通过SIGKILL信号强制删除一个运行中的容器
  • -l 移除容器间的网络连接
  • -v 删除容器关联的卷
  • -d 后台运行
  • -i 交互模式运行
  • -t 分配一个伪终端

容器操作

  • -a 显示所有容器
  • -f 根据条件过滤显示的内容
  • –format 指定返回值的模板文件
  • -l 显示最近创建的容器
  • -n 列出最近创建的n个容器
  • –no-trunc 不截断输出
  • -q 静默模式，只显示容器编号
  • -s 显示总的文件大小
  • -f 指定返回值的模板文件
  • -s 显示总的文件大小
  • –type 为指定类型返回json
  • -f 跟踪日志输出
  • -t 显示时间戳
  • –tail 仅列出最新N条日志
  • –since 显示某个开始时间的日志

容器rootfs命令

  • -a 提交镜像的作者
  • -c 使用dockerfile指令来创建镜像
  • -m 提交时的说明文字
  • -p 在commit时将容器暂停
  • -L 保持源目标中的链接

本地镜像管理

  • -a 列出本地所有镜像
  • –digests 显示摘要信息
  • -f 显示满足条件的镜像
  • –format 指定返回值的模板文件
  • –no-trunc 显示完整镜像信息
  • -q 只显示镜像ID
  • -f 强制删除
  • –no-prune 不移除过程镜像
