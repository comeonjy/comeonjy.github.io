Golang中defer、return和返回值

执行顺序

  1. return
  2. defer
  3. 函数返回

return

  • 将return后面的表达式的值赋给返回值，若为匿名返回值，则赋值给一个临时变量，执行完defer后再将其返回

defer

  • 作用域：只对当前协程有效
  • 程序panic后依然会执行defer，主动调用os.Exit(int)不会执行defer
  • defer声明时会先计算参数的值
  • 多个defer执行顺序为先进后出
